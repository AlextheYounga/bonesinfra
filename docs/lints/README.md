# Linter Documentation

This directory contains documentation and integration prompts for the agentlinters.

## Integration Prompt

Use the `integration-prompt.md` file in each language subdirectory as a prompt when asking an AI agent to help integrate these linters with an existing project.

## Quick Start

1. Run `agentlinters` to install linter configs
2. Copy the `docs/lints/{language}/integration-prompt.md` content for the language you installed
3. Ask an AI agent to integrate based on this prompt
4. The agent will review the installed configs and help fix violations
